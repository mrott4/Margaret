Learning to Code I - San Leandro Adult School

Student: Margaret Rott

I learned many new things in the Learning to Code I class. Here are are some of the highlights.

  • Coding is like using an alphabet to write, except you're using the unique set of characters and
    syntax of a given programming language.

  • All documents that appear in a web browser have at least the following three Hypertext
    Markup Language (HTML) elements: <!DOCTYPE html>, <html>-</html> and <body>-
    </body>. These elements provide the basic structure for the browser content.

  • Cascading Style Sheets (CSS) tell the browser how to display the content. For example, CSS
    can be coded for the background color, the font family, or a list format.

  • This class reinforced the importance of setting goals and tracking achievements. Each week
    we reported our objectives and outcomes for the day in a pull request on GitHub, a system for
    managing software development.

  • Adding comments to code and GitHub pull requests can provide valuable information about
    why the coding was done as it was.

  • My Meetup with the Women Who Code East Bay-Programming and Web Development was
    cancelled. In lieu of this meetup, I listened to a recording of a recent meeting of the Silicon
    Valley Chapter of the Society for Technical Communication. The speaker was from Microsoft
    and spoke on open-source systems, GitHub in particular. He made the point that in the current
    agile environment, technical writers are hired first for their technical expertise and
    secondarily for their writing ability. It is assumed, in fact, that all staff have at least some
    writing skill, but their technical expertise is what really counts.

As our Learning to Code I class comes to a close, I don't know if I'll be able to continue with a
class next term. I do know that there will be more coding in my future, whether in the classroom
or on my own, or who knows - maybe on the job! Thank you, San Leandro Adult School and
especially our instructor Lisa Rose, for a very worthwhile experience.
